# Oracle Database 테이블 리스트 및 정보

## 개요
이 디렉토리는 정보보안기사 시험 준비를 위한 Oracle Database 테이블 구조 및 데이터 딕셔너리 정보를 포함합니다.

## 디렉토리 구조
- `system_tables.md` - Oracle 시스템 테이블 목록
- `data_dictionary_views.md` - 데이터 딕셔너리 뷰 목록
- `sample_schemas.sql` - 샘플 스키마 생성 스크립트
- `security_tables.md` - 보안 관련 테이블 및 뷰
- `dba_views.md` - DBA 관련 뷰 목록
- `user_views.md` - 사용자 레벨 뷰 목록
- `performance_views.md` - 성능 관련 뷰 목록
- `audit_tables.md` - 감사(Audit) 관련 테이블 및 뷰

## 주요 내용
### 1. 데이터 딕셔너리
Oracle 데이터 딕셔너리는 데이터베이스의 메타데이터를 저장하는 읽기 전용 테이블 및 뷰의 집합입니다.

### 2. 시스템 테이블
Oracle의 핵심 시스템 테이블들은 SYS 스키마에 저장되며, 데이터베이스의 기본 구조를 정의합니다.

### 3. 보안 관련 테이블
사용자 권한, 역할, 감사 로그 등 보안과 관련된 정보를 저장하는 테이블 및 뷰입니다.

## 참고사항
- 정보보안기사 시험에서 데이터베이스 보안은 중요한 과목입니다
- Oracle의 권한 관리, 감사 기능, 암호화 등을 이해하는 것이 중요합니다
- 실습을 위해서는 Oracle Database Express Edition(XE)을 사용할 수 있습니다
