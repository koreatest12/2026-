# Oracle 시스템 테이블 목록

## 주요 시스템 테이블 (SYS 스키마)

### 핵심 시스템 테이블

| 테이블명 | 설명 |
|---------|------|
| TAB$ | 모든 테이블의 기본 정보 저장 |
| COL$ | 모든 컬럼의 정보 저장 |
| USER$ | 데이터베이스 사용자 정보 |
| OBJ$ | 모든 데이터베이스 객체 정보 |
| IND$ | 인덱스 정보 |
| SEG$ | 세그먼트 정보 |
| TS$ | 테이블스페이스 정보 |
| FILE$ | 데이터파일 정보 |
| UET$ | 사용된 익스텐트 정보 |
| FET$ | 사용 가능한 익스텐트 정보 |

### 보안 관련 시스템 테이블

| 테이블명 | 설명 |
|---------|------|
| USER$ | 사용자 계정 정보 (암호화된 패스워드 포함) |
| PROFILE$ | 프로파일 정보 |
| RESOURCE$ | 리소스 제한 정보 |
| DEFROLE$ | 기본 역할 정보 |
| SYSAUTH$ | 시스템 권한 정보 |
| OBJAUTH$ | 객체 권한 정보 |
| LINK$ | 데이터베이스 링크 정보 |

### 감사 관련 시스템 테이블

| 테이블명 | 설명 |
|---------|------|
| AUD$ | 표준 감사 레코드 |
| FGA_LOG$ | Fine-Grained Auditing 로그 |
| AUDIT_ACTIONS | 감사 액션 코드 매핑 |

### 스토리지 관련 시스템 테이블

| 테이블명 | 설명 |
|---------|------|
| TABLESPACE$ | 테이블스페이스 정보 |
| DATAFILE$ | 데이터파일 정보 |
| TEMPFILE$ | 임시파일 정보 |
| UNDO$ | UNDO 세그먼트 정보 |

### 제약조건 관련 시스템 테이블

| 테이블명 | 설명 |
|---------|------|
| CDEF$ | 제약조건 정의 |
| CCOL$ | 제약조건 컬럼 |
| CONS$ | 제약조건 정보 |

### 트리거 관련 시스템 테이블

|---------|------|

### 시퀀스 관련 시스템 테이블

|---------|------|
| SEQ$ | 시퀀스 정보 |

### 뷰 관련 시스템 테이블

| 테이블명 | 설명 |
|---------|------|
| VIEW$ | 뷰 정의 정보 |
| VIEWCOL$ | 뷰 컬럼 정보 |

### 프로시저/패키지 관련 시스템 테이블

| 테이블명 | 설명 |
|---------|------|

## 참고사항
- 시스템 테이블은 직접 수정하면 안 됩니다
- 시스템 테이블은 SYS 사용자만 접근 가능합니다
- 일반 사용자는 데이터 딕셔너리 뷰를 통해 정보를 조회해야 합니다
